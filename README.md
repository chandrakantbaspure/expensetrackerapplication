## Setup

To set up this project locally:

1. Clone the repository:
   Git URL:  https://github.com/chandrakantbaspure/expensetrackerapplication.git
2. Install dependencies:
   mvn clean install
3. Start the application:
   mvn spring-boot:run
4. Start the application
   server.port=8082
